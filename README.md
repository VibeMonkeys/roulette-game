# 🎰 레트로 슬롯머신 게임

1999년 감성의 회사 이벤트용 슬롯머신 웹앱입니다. 모듈화된 ES6 아키텍처로 구축되어 유지보수와 확장이 용이합니다.

## 🎮 게임 방법
- **SPIN** 버튼 클릭 또는 **스페이스바**로 게임 시작
- 3개의 릴이 순차적으로 정지하며 결과 표시
- 사용자당 **1회만 참여 가능**
- 모바일 터치 지원 (갤럭시, 아이폰 호환)

## 🏗️ 프로젝트 구조
```
roulette-game/
├── index.html          # 메인 HTML 파일
├── style.css           # 1999년 레트로 스타일
├── PRD.md             # 제품 요구사항 문서
└── src/               # 모듈화된 소스코드
    ├── main.js            # 메인 엔트리 포인트
    ├── core/             # 게임 핵심 로직
    │   ├── SlotMachineEngine.js  # 게임 엔진
    │   └── GameState.js          # 상태 관리
    ├── ui/               # UI 컴포넌트
    │   ├── UIManager.js         # UI 관리자
    │   └── AnimationSystem.js   # 애니메이션 시스템
    ├── systems/          # 게임 시스템
    │   ├── ProbabilitySystem.js # 확률 & 당첨 계산
    │   └── SaveSystem.js        # 저장/로드 시스템
    └── utils/            # 유틸리티
        ├── Constants.js         # 상수 정의
        └── Helpers.js          # 헬퍼 함수들
```
